# LeetCode Journey

> A personal algorithm journey: record, practice, and grow step by step.

## About This Repository

This repository is used to archive my algorithm learning notes and coding solutions.
I will continuously upload solved problems, ideas, and summaries during my LeetCode journey.

Even though the project is at an early stage, it will gradually become a structured and practical handbook for interview preparation.

## Goals

- Build a long-term record of solved algorithm problems
- Keep code, notes, and thoughts in one place
- Improve problem-solving speed and coding quality
- Prepare for technical interviews in a systematic way
