# Solutions

This folder stores solved problems by language.

- `python/`: Python solutions
- `cpp/`: C++ solutions
- `java/`: Java solutions

Naming suggestion:

`<problem_id>_<short_title>.<ext>`

Example:

- `0001_two_sum.py`
- `0206_reverse_linked_list.cpp`

Tip:

Use `templates/problem_template.*` to keep a consistent format for every solution file.
