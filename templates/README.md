# Templates

This folder stores reusable algorithm templates.

Current templates:

- Binary Search
- DFS / BFS
- Backtracking
- Union Find
- Prefix Sum

Problem file templates:

- `problem_template.py`
- `problem_template.cpp`
- `problem_template.java`

How to use:

1. Copy one template to `solutions/<language>/`.
2. Rename the file, for example: `0121_best_time_to_buy_and_sell_stock.py`.
3. Replace placeholders: `{id}`, `{title}`, `{slug}`, `{topic}`.
4. Fill in idea and complexity.
